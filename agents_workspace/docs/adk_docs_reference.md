## What ADK includes (from official docs)
- ADK is presented as a framework to build, evaluate, and deploy AI agents.
- Core agent model includes:
  - LLM agents (`LlmAgent`/`Agent`)
  - Workflow agents (`SequentialAgent`, `ParallelAgent`, `LoopAgent`)
  - Custom agents by extending `BaseAgent`
- Runtime model is event-loop based, orchestrated by ADK runner/runtime components.
- Context handling is built around `Session`, `State`, and `Memory`.
- Tooling includes pre-built tools, function tools, MCP-based integrations, and OpenAPI-based tool generation.
- Evaluation is supported via:
  - `adk web` UI
  - programmatic/`pytest` flow
  - CLI `adk eval`
- Deployment paths include Vertex AI Agent Engine, Cloud Run, GKE, and other container-compatible infrastructure.
- ADK docs also include observability integrations and tracing ecosystem guides.
